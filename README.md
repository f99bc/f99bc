# Página de presentación personal

Sitio estático creado con HTML, CSS y JavaScript, listo para publicarse con GitHub Pages.

## Archivos

- `index.html`: estructura y contenido.
- `styles.css`: diseño, colores y adaptación a dispositivos móviles.
- `script.js`: menú móvil, cambio de tema, año automático y animaciones.

## Personalización rápida

Busca y reemplaza en `index.html`:

- `Tu Nombre`
- `TU_USUARIO`
- `correo@ejemplo.com`
- Las descripciones de habilidades y proyectos
- Los enlaces `href="#"` de cada proyecto

## Publicación

En GitHub:

1. Abre **Settings**.
2. En el menú lateral abre **Pages**.
3. En **Source**, selecciona **Deploy from a branch**.
4. Selecciona `main` y `/(root)`.
5. Pulsa **Save**.
